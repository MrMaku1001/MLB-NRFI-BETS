📐 How Odds Are Derived
1
Model estimate — Poisson formula: P(0 runs) = e^(−λ), λ from pitcher ERA + team offense
2
Live FD odds — When your API key is set, real FanDuel game total (over/under) is fetched and used to derive a tighter NRFI estimate using the Skellam distribution
⚡
Edge indicator — Shows when model probability differs from FanDuel's implied probability by 5%+. Green = model says better odds than FD is offering
